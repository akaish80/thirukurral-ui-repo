# Repository Agents

This repository defines two AI helper agents under .github/agents:

1. Code Review Assistant
- File: .github/agents/code-review.agent.md
- Focus: quality gates, accessibility, API stability, tests, Storybook coverage.

2. Vercel Deploy Assistant
- File: .github/agents/vercel-deploy.agent.md
- Focus: deployment readiness, environment validation, Vercel workflow health.

Use these as standard prompts for GitHub Copilot Chat, local coding agents, or repository automation playbooks.
